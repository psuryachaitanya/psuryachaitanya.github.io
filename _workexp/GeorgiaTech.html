Software Web Developer   [ React, Spring Boot, Azure Web Apps, Azure Devops]

  • Conceptualized, designed, and developed a dynamic full-stack web application, resulting in a 30% reduction in request processing time and an increase in collaboration efficiency between administrative staff and faculty through a comprehensive ticketing system.
  • Implemented OpenID Connect to allow a single sign-on experience for users of the application.
  • Spearheaded the implementation of an end-to-end DevOps pipeline and enabled seamless integration, automated testing, and continuous deployment using Azure DevOps Pipelines, Azure Web App for containers, Git, and Docker.
